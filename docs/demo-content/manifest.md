# Демо-контент для seed-наполнения платформы

Восемь роликов общим объёмом ~66 МБ для первичного засева платформы. Все легальные: шесть Blender Open Movies (CC BY 3.0/4.0), два интернет-мема (Trololo — публичный эфир Гостелерадио СССР 1976; Coffin Dance — съёмка BBC Africa 2017, вирально перевернутая в COVID-эру).

Семь из восьми со звуком (AAC). `bbb-4k.mp4` без аудио — это естественно для 4K-демо. Загружать под основной аккаунт ведущего, чтобы профиль выглядел заполненным.

## Источники

- **Big Buck Bunny** — Blender Foundation, 2008, CC BY 3.0. Трейлер с `download.blender.org/peach/trailer/`; 4K-фрагмент с YouTube-канала Blender Foundation, обрезан до 12 секунд.
- **Sintel** — Blender Foundation, 2010, CC BY 3.0. Трейлер с `download.blender.org/durian/trailer/`.
- **Caminandes 3: Llamigos** — Blender Foundation, 2016, CC BY. Видео с официального канала Blender на YouTube.
- **Spring** — Blender Animation Studio, 2019, CC BY. Видео с официального канала Blender.
- **Charge** — Blender Animation Studio, 2022, CC BY. Видео с официального канала Blender.
- **Trololo** — выступление Эдуарда Хиля в передаче Гостелерадио СССР, 1976; с 2010 — известный интернет-мем.
- **Coffin Dance** — съёмка похоронной церемонии в Гане, агентство BBC Africa, 2017; в 2020 году стала глобальным мемом эпохи коронавируса.
